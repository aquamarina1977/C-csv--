# C-csv--
Скрипт читает файлы с данными о закрыты задачах и формирует отчеты. Отчёт включает в себя список позиция и среднюю эффективность (среднее арифметическое по колонке performance). Название файлов и название отчета передается в виде параметров --files и --report. Отчёт формируется по всем переданных файлам, а не по каждому отдельно.
